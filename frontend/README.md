
# Push and Pray - Full React Frontend

This is a complete frontend implementation built with:

- React
- Vite
- React Router

## Pages included
- Login / Sign up
- Dashboard
- Repositories
- Activity
- Team
- Profile

## Run locally

```bash
npm install
npm run dev
```

Then open the local address from the terminal, usually:

```bash
http://localhost:5173
```

## Notes
- Authentication is demo-based via localStorage
- The UI is ready to be connected to a backend later
- No database is required for the frontend to open
